About This System

Our Facial Recognition Attendance System leverages cutting-edge technology to revolutionize how organizations track attendance. By combining Django's robust backend with OpenCV's powerful LBPH algorithm, we've created a solution that is both accurate and efficient.

The system captures facial images through the browser, processes them securely, and stores the data as BLOB in MySQL databases. This approach ensures data integrity while maintaining high performance for attendance matching.

With a focus on user experience and security, our system provides a seamless way to manage attendance without the need for physical cards, fingerprints, or manual sign-in sheets.

System Features

Real-time Face Capture

Secure Data Storage
