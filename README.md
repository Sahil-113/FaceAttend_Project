# FaceAttend_Project

FaceAttend_Project is an automated attendance system leveraging facial recognition technology. This project aims to provide a robust and contactless method for tracking attendance, suitable for academic institutions, workplaces, and events.

## Features

- **Facial Recognition:** Uses computer vision techniques to detect and identify faces.
- **Attendance Automation:** Streamlines attendance marking with minimal human intervention.
- **User-Friendly Interface:** Easy to use for administrators and end-users.
- **Secure Data Handling:** Ensures privacy and security of user data.

## Technologies Used

- **Python** (core logic and backend)
- **OpenCV** (for image processing and face detection)
- **Django** (optional - for web application interface)
- **MySQL** (for data storage)

## Getting Started

### Prerequisites

- Python 3.x
- OpenCV

### Installation

```bash
git clone https://github.com/Sahil-113/FaceAttend_Project.git
cd FaceAttend_Project
# Install required libraries
pip install -r requirements.txt
```

### Usage

1. Add reference images for each user in a designated folder.
2. Run the main script to start facial recognition and attendance logging.
3. View or export attendance reports as needed.
